# Parser application
The application which parse .csv and .json files 
and on the output gives calculated salary for specific jobs for each file.



## Requirements
* Java 11+
* Maven

##How to build
Run and build as every maven project.

```bash
# How to run test
mvn clean test

# and next you should

# How to install
mvn clean install
```
